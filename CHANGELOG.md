# PA Changelog - 版本记录

## v1.0.0 (2026-04-03) - 正式发布

### 🎨 界面美化
- **渐变主题**：用户消息气泡采用渐变背景 + 闪烁动效
- **玻璃态效果**：导航栏、侧边栏、下拉菜单增加模糊和半透明效果
- **发光效果**：按钮、头像、主题切换等交互元素添加发光动画
- **渐变指示器**：导航链接、底部 Tab 添加渐变下划线/指示条
- **登录页重做**：悬浮动画 Logo、渐变按钮、输入图标、加载动画

### 🗣️ 语音输入功能
- 在 Chat 页面添加麦克风按钮
- 使用 Web Speech API 实现语音识别
- 支持中文识别，结果自动填充到输入框
- 录音时按钮红色脉冲动画

### 🤖 PA 网页版 AI 聊天修复
- 创建 `aiService.js` 直接调用 KAT-Coder 模型 API
- 支持流式响应（SSE）
- 自动从数据库加载对话历史作为上下文
- 上下文窗口限制内保留重要对话记录
- 不再依赖 OpenClawWS WebSocket 连接

### 🐛 问题修复
- 修复 Gateway 端口配置（3000 → 18789）
- 禁用 OpenClawWS WebSocket 连接（设备配对问题）
- 修复 PA 后端所有 API 连接地址
- 修复新建对话标题显示 fallback

### 🧹 配置清理
- 移除微信插件（openclaw-weixin）
- 移除 MiniMax 模型提供商
- 移除 zai 模型提供商
- 清理 auth profiles 中的无效配置

### 📋 待办功能
- 新增语音输入功能（#49）
- 确认开场白功能已存在（#50）
- 确认标题显示正常（#51）
- 完成全量 API 测试（#52）

### 🔧 技术改进
- 主题变量增强：添加 `--accent-gradient`、`--accent-glow` 等
- 添加 `--font-mono` 和 `--font-sans` 字体变量
- 优化移动端响应式体验
- 添加内容淡入动画
